## Project Description

This project demonstrates the creation of a basic ERC-20 style token using Solidity. The smart contract defines a custom cryptocurrency with its own token name, symbol, decimals, total supply, and token transfer functionality.

## Technologies Used

* Solidity
* Remix IDE
* Ethereum Virtual Machine (Remix VM)

## Features

* Custom Token Name
* Custom Token Symbol
* Total Supply Generation
* Token Balance Management
* Token Transfer Functionality
